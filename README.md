# 妙笔科技 AI 营销平台

## 快速部署到 Vercel

### 一键部署

1. 访问 https://vercel.com 注册账号
2. 将此文件夹推送到 GitHub
3. 在 Vercel 点击 "New Project"
4. 选择你的仓库，点击 "Deploy"
5. 等待 1-2 分钟 → 获得永久免费网址！

## 项目说明

- **技术栈**: Next.js 14 + React 18 + Tailwind CSS
- **设计风格**: 苹果式极简 + 赛博深邃元素
- **品牌**: 妙笔科技

## 本地开发

```bash
npm install
npm run dev
```

访问 http://localhost:3000
