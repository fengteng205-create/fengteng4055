## 🌐 如何让网站全网可用？

### 方案一：临时外网访问（立即可用，5分钟搞定）

#### 使用 LocalXpose（推荐，完全免费，无需注册）
1. 下载: https://localxpose.io/download
2. 解压后打开命令行，运行:
   ```bash
   loclx tunnel http --to localhost:3001
   ```
3. 复制生成的公网网址，分享给任何人！

#### 使用 Serveo（最简单）
```bash
ssh -R 80:localhost:3001 serveo.net
```

### 方案二：永久免费部署（推荐长期使用）

#### Vercel 部署（最简单，对 Next.js 支持最好）
1. 注册 Vercel: https://vercel.com
2. 将 `AI营销平台` 文件夹推送到 GitHub
3. 在 Vercel 点击 "New Project"
4. 选择你的仓库，点击 "Deploy"
5. 等待 1-2 分钟 → 获得永久免费网址！

#### Cloudflare Pages（中国访问速度快）
1. 注册: https://pages.cloudflare.com
2. 创建项目，连接 GitHub
3. 配置:
   - 构建命令: `npm run build`
   - 输出目录: `.next`
4. 点击部署 → 完成！
