# 妙笔科技 - 永久部署指南

## 🎯 目标：完全无确认的永久访问

### 方案一：Vercel 部署（推荐，最简单）

#### 步骤 1：注册 GitHub 账号
1. 访问 https://github.com
2. 点击 "Sign up" 注册账号（免费）
3. 验证邮箱并登录

#### 步骤 2：创建新仓库
1. 登录 GitHub 后，点击右上角 "+" → "New repository"
2. 仓库名称：`miaobikeji`（或您喜欢的名字）
3. 选择 "Public" 或 "Private" 都可以
4. 点击 "Create repository"

#### 步骤 3：上传代码
1. 打开您的 `AI营销平台` 文件夹
2. 选中所有文件和文件夹（包括 app、components、lib、package.json 等）
3. 拖放到 GitHub 仓库页面
4. 点击 "Commit changes"

#### 步骤 4：部署到 Vercel
1. 访问 https://vercel.com
2. 点击 "Sign Up"，用 GitHub 账号登录
3. 点击 "New Project"
4. 选择您刚创建的 `miaobikeji` 仓库
5. 点击 "Deploy"
6. 等待 1-2 分钟，部署完成！

#### 完成！
您会获得一个永久免费的网址，例如：`https://miaobikeji.vercel.app`

---

### 方案二：Cloudflare Pages（中国访问更快）

#### 步骤 1-3：同方案一（上传到 GitHub）

#### 步骤 4：部署到 Cloudflare Pages
1. 访问 https://pages.cloudflare.com
2. 注册账号（免费）
3. 点击 "Create a project" → "Connect to Git"
4. 选择 GitHub 仓库
5. 配置：
   - 构建命令：`npm run build`
   - 输出目录：`.next`
6. 点击 "Save and Deploy"

#### 完成！
获得永久免费网址，中国访问更快！
